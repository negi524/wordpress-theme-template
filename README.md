# wordpress-theme-template

ここからカスタマイズを行う前提の、WordPress テーマテンプレート

## 開発方法

### 1. プロジェクトのインストール

```bash
npm install
```

### 2. 開発時(ホットリロードあり)

```bash
npm run dev
```

### 3. コードフォーマット

prettier によるコードフォーマットを適用する。

```bash
npm run format
```

### 4. 本番環境用のファイルをビルド

```bash
npm run build
```
